Arguments

Arguments are accessed via the positional parameters $1, $2, Â£3 etc. For example

# printer.sh

#!/bin/bash
echo "Parameter one: $1"
echo "Parameter two: $2"

results in

$ ./printer.sh foo bar
Parameter one: foo
Parameter two: bar

when executed.

There are also special positional parameters. Some examples include:

  • $# - the number of positional parameters
  • $@ - an array of all positional parameters
  • $- - current shell options

For a handy list of variables see [1]. For a more exhaustive list see [2]. A list with example outputs can be found here [3].
